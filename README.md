
# Integrated Application Development Framework : 
**Charity Management System**

A **web application** built with **Laravel** to manage donors, donations, and user accounts. The app uses **MVC architecture** 

## Features

* **User Authentication**

  * Registration
  * Login
  * Reset password
* **Donor Management**

  * Add, update, delete donors
  * View donor lists
* **Donation Management**

  * Record donations
  * Track donation history
* **API Support**

  * RESTful endpoints for donors and donations
* **MVC Structure**

  * Models, Views, Controllers separated for clean code

## Technologies

* **Framework:** Laravel
* **Database:** MySQL 
* **Frontend:** Blade templates
* **API:** RESTful JSON responses

## Installation

1. **Clone the repository**

```bash
git clone 
cd charity-management-system
```

2. **Install dependencies**

```bash
composer install
npm install
npm run dev
```

3. **Configure environment**

* Copy `.env.example` to `.env`
* Set database credentials
* Generate application key

```bash
php artisan key:generate
```

4. **Run migrations**

```bash
php artisan migrate
```

5. **Start the server**

```bash
php artisan serve
```

6. **Access the app**

Open localhost in your browser.
